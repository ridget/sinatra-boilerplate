#	Sinatra Boilerplate

Sinatra, Datamapper, and Rspec in an easy to use template.

##	Getting Started

Clone

	git clone git@github.com:theidealab/sinatra-boilerplate.git

Remove this remote

	git remote rm origin

Add your own remote

	git remote add origin git@github.com:my_org/my_repo.git

Start working~
